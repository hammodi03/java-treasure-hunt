# Treasure Hunt Game

A desktop treasure hunt game developed in Java using Swing and the MVC architectural pattern.

Players take turns digging for hidden treasures while avoiding traps. The game includes multiple treasure and trap types, score tracking, persistent high scores, and an interactive graphical user interface.

## Features

- Java Swing GUI
- MVC Architecture
- Object-Oriented Design
- Multiple Treasure Types
- Trap System
- Turn-Based Gameplay
- Persistent High Scores
- File Handling
- UML-Based Design

## Technologies

- Java
- Swing/AWT
- MVC
- Object-Oriented Programming
- File I/O
